Gedima'Gination

Gedima'Gination est un concours de photo fictif, organisé par la société Gedimat, pour laquelle nous devions développer deux solutions :

  • Une Application Web
  • Une Application Android

Pour participer au concours, les personnes se rendent sur l'application web où ils peuvent y creer un compte et selectionnent ensuite la photo qu'ils veulent mettre en competition. En effectuant un achat en magasin un client obtient un ticket qu'il peut utiliser sur l'application Android en magasin pour voté pour ses trois réalisations préférées. A la fin d'une date déterminé le classement est révélé sur l'application web et les trois premier remportent un bon d'achat.

Application Web

L'application web permet de s'inscrire et de participer au concours. Le gestionnaire peut fixer les dates d'inscription, de participation et de fin de votes. Lorsqu'il le souhaite il peut déployer le classement finale qui sera visible par tous le monde y compris "les simples visiteurs".

Application Android

L'application Android permet de au client, une fois avoir reçus leur ticket, de voter pour leur trois réalisations préférées. Une fois la période de votes terminée, le gestionnaire peut envoyer les données des votes via une API afin de les afficher sur l'application web.

Gestion du travail

Ce projet a été réalisé en équipe constitué de Levasseur Guillaume et moi-même.

Ce projet a été réalisé dans un cadre fictif. Nous devions suivre une démarche agile, sous forme de sept sprints d'une semaine.

Pour cet exercice, nous avons fais le choix de se répartir sur les technologies que l'on préférait. Guillaume s'occupait de l'application Android et moi du Web. Cela nous a ainsi permis d'avancer plus vite sans devoir travailler sur une technologie que l'on apprécie moins.

Technologies Utilisées

  • JavaScript

  • GitHub

Résultat

Conclusion

Les deux applications sont complètes et remplissent le cahier des charges. Malheuresement il nous manque seulement la fonctionalité d'envoyer les réalisations de l'application Web vers l'application Android.

Ce projet m'aura permis d'appréhender ce qu'est une gestion de projet agile ainsi que d'augmenter mes connaissances et compétences dans le Web et pour le langage PHP.
