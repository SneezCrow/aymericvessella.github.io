Laravel

Qu'est ce que Laravel ?

Laravel est un framework PHP permettant de créer des applications web et utilise un langage de script plutôt qu’un langage de programmation PHP strict. Bien que les langages de script et les langages de programmation soient apparentés, ils présentent plusieurs différences notables, principalement en termes de facilité d’utilisation et de vitesse d’exécution.

Pourquoi Laravel ?

J'ai choisi de faire ma veille sur Laravel car je m'interessais au langage PHP et on m'a parlé de ce framework.Il est très populaire dans le monde avec une grande communauté et beaucoup de documentation. Malgré cela, en France, le framework le plus utilisé est Symfony.
